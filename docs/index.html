Sobre o Projeto

Netflixo é uma galeria de vídeos desenvolvida como parte da aula do professor Alan Veloso. Este projeto oferece uma interface moderna e inspirada no Netflix, permitindo que os usuários visualizem e naveguem por uma coleção de vídeos de forma organizada e intuitiva.

A aplicação utiliza HTML, CSS e JavaScript para criar uma experiência de usuário responsiva e agradável. A principal funcionalidade é a reprodução de um vídeo por vez, garantindo que o conteúdo seja exibido de forma clara e sem sobreposição.
